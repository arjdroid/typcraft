# typcraft

[typcraft](https://typcraft.nxtdroid.win) is a website to practice composing equations in the [typst](https://typst.app) typesetting system. It takes inspiration from [texnique.xyz](https://texnique.xyz) which provides a fun way to do the same in LaTeX.
